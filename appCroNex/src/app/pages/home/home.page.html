Bienvenido a Cronex

CRONEX está contemplada como una aplicación híbrida (Mobile/Web), utilizando un FrameWork de desarrollo llamado Ionic para su Front-End, acompañado de tecnologías como HTML, ANGULAR, TS (TypeScript), SCSS; en su Back-End contendrá tecnologías como JSON y como gestor de BD (Base de datos) se utilizará FireBase una herramienta que nos proporciona Google.
